## 项目描述

Reply 是一款多屏幕应用，类似于[电子邮件客户端](https://en.wikipedia.org/wiki/Email_client)。该应用包含 4 个不同类别，分别显示在不同的标签页中，即 “Inbox” 、“Sent”、 “Drafts”和“Spam”。
