### 实现项目

- [MarsPhotos](MarsPhotos\docs\README.md)：采用 `Retrofit` 获取互联网数据，并利用 `kotlinx.serialization` 转换器将 JSON 对象转换为 Kotlin 对象，同时使用 `Coil` 下载和显示图片。同时，了解如何将代码设计成界面层和数据层，并使用容器和 `Application` 类来实现依赖注入
