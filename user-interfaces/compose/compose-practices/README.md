## 专题总览

Compose 官方实践案例，对应 [Android Basics with Compose](https://developer.android.google.cn/courses/android-basics-compose/course?hl=zh-cn) 课程路径。



### 项目结构

- [HelloWorld](HelloWorld\docs\README.md)：创建 Hello World 项目，了解 Android 项目的基本构建流程
- [HappyBirthday](HappyBirthday\docs\README.md)：构建生日贺卡应用，学习如何添加文本和图片
- [DiceRoller](DiceRoller\docs\README.md)：创建掷骰子应用，了解如何在 Android 应用中响应按钮点击
- [TipCalculator](TipCalculator\docs\README.md)：小费计算器，了解如何使用状态来显示数据并在数据更新时自动反映更改
- [Affirmations](Affirmations\docs\README.md)：自我肯定话语列表，学习 Card、LazyColumn 与自适应图标
- [Unscramble](Unscramble\docs\README.md)：猜词游戏，了解 ViewModel、UI State 与 StateFlow（亦见 [lifecycle-aware](../../../app-architecture/lifecycle-aware/README.md)）
- [Cupcake](Cupcake\docs\README.md)：购买蛋糕应用，学习 Compose Navigation 与多屏导航（亦见 [app-navigation](../../../app-architecture/app-navigation/README.md)）
- [Reply](Reply\docs\README.md)：邮件客户端风格应用，学习自适应布局与多尺寸屏幕
