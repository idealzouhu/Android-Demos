## 项目描述

[本项目](https://developer.android.google.cn/courses/pathways/android-basics-compose-unit-2-pathway-3?hl=zh-cn)构建一款 Tip Time 应用，它会根据用户输入的账单金额和小费百分比计算小费金额。



## 学习内容

- 了解状态，以及 Jetpack Compose 如何使用和操纵状态。
- 了解 `remember()` 函数、状态、状态提升，以及有状态和无状态可组合函数之间的差异。
- 能够向应用添加 `Text` 和 `TextField` 可组合项，向文本字段添加前置图标。

- 了解如何添加操作按钮、设置键盘操作以及使用 Switch 可组合项。
