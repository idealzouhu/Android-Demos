### 实现项目

- [Cupcake](..\..\ui-frameworks\compose-basics\Cupcake\README.md): 创建一款购买蛋糕的应用，了解如何使用 Navigation 组件构建包含更多屏幕的更复杂的应用，以及如何在不同屏幕之间进行导航和数据传递。
- [fragment-responsive-java-view 项目实现](.\fragment-responsive-java-view\README.md): 演示了 Android 中单 Activity 多 Fragment 的响应式布局用法，重点展示如何根据屏幕宽度（手机、平板、大屏）切换不同导航方式：底部导航栏、侧边抽屉与常驻侧边栏，以及溢出菜单的按需显示。
- [viewpage2-basic-java-view 项目实现](docs\viewpage2-basic-java-view 项目实现.md): 利用 ViewPage2 实现带标签的滑动视图（水平分页），配合 TabLayout 与 FragmentStateAdapter 完成多页切换与联动。
